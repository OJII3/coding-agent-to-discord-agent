# coding agent to discord agent

コーディングエージェントをDiscordエージェントとして使うためのツールです。
opencode や openclaw のようなサードパーティーツールではなく、CLIをラップしています。

- claude code
- codex cli
- gemini cli
- copilot cli

のようなCLIコーディングエージェントでの使用を想定しています。

## 特徴

公式CLIツールをそのまま使用します。

メリット
- 非公式クライアントツールを使用するグレーな方法を避けることができます
- 手軽に ai discord bot を構築できます

デメリット
- 公式CLIツールの制約を受けるため、機能や性能が制限される可能性があります
- 公式CLIツールの利用規約に従う必要があるため、使用に制限がある場合があります(各自の責任で使用してください)
- 公式CLIツールの更新により、エージェントの動作が影響を受ける可能性があります
- 公式CLIツールの内部設定やプロンプトにより、性格や応答のスタイルが制限される可能性があります
